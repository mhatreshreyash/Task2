# 🎓 Student Grading Program (Python)

## 📌 Project Description

This is a simple Python program that takes a student's name and marks as input, validates the marks, assigns a grade (A, B, C, D, F), and displays an encouraging message based on performance.

The program is beginner-friendly and demonstrates core programming concepts like functions, conditional statements, loops, and input validation.

---

## 🚀 Features

* Accepts student name and marks as input
* Validates marks (only allows values between 0–100)
* Uses `if-elif-else` for grading logic
* Includes a `while` loop to handle invalid inputs
* Uses a function to determine grade and message
* Displays encouraging messages for each grade

---

## 🛠️ Technologies Used

* Python (Basic concepts)

---

## 📋 Grading Criteria

| Marks Range | Grade |
| ----------- | ----- |
| 90 – 100    | A     |
| 75 – 89     | B     |
| 60 – 74     | C     |
| 50 – 59     | D     |
| Below 50    | F     |
